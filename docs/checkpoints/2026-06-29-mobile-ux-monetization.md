쩌렁쩌렁(Flutter `mobile/`) 만화 만들기·더빙·수익화 전반 개선 세션 요약.

## 1. 인물(캐릭터) 생성·적용 플로우 정리
- AI 스토리보드에 **화자별 캐릭터 배정** 추가 → 컷마다 맞는 인물로 일괄 생성(이름 자동 매칭 + 그 자리에서 생성). `ai_studio.dart` `_StoryboardSheet`, `creator_screen.dart` `_generateAllCuts`
- 만화 만들기 화면 등장인물 섹션을 **목록 표시 전용**으로(생성·'기본 선택' 제거). 인물 없으면(=사진만) 섹션 숨김 → 사진 모드엔 인물 개념 비노출
- 인물 생성은 **필요한 순간**(AI 컷 시트·스토리보드 화자칸)에서만
- 커밋: `014af63`, `5f9233e`

## 2. 발행물 수정 기능

- 진입: 내 작업 목록 + 더빙 화면(작가 본인 `canEdit`)
